# To My Favorite Person

This website is dedicated to my favorite person, to let them know how much value they brought to my life

[![My Skills](https://skillicons.dev/icons?i=css,ts,tailwind,react,nextjs)](https://skillicons.dev)

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/jane-does-coding/ToMyFavoritePerson
cd to-my-favorite-person
```

### 2. Install the dependencies

```bash
npm i
```

### 3. Run the project

```bash
npm run dev
```
